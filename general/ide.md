# Adding `.vscode/settings.json` files

Customize colors per project so tabbing is easier!
```
{
  "editor.formatOnSave": true,
  "workbench.colorCustomizations": {
    "activityBar.activeBackground": "#feb12d",
    "activityBar.background": "#feb12d",
    "activityBar.foreground": "#15202b",
    "activityBar.inactiveForeground": "#15202b99",
    "activityBarBadge.background": "#5c70e3",
    "activityBarBadge.foreground": "#e7e7e7",
    "commandCenter.border": "#15202b99",
    "sash.hoverBorder": "#feb12d",
    "statusBar.background": "#feb12d",
    "statusBar.foreground": "#15202b",
    "statusBarItem.hoverBackground": "#d97706",
    "statusBarItem.remoteBackground": "#feb12d",
    "statusBarItem.remoteForeground": "#15202b",
    "titleBar.activeBackground": "#feb12d",
    "titleBar.activeForeground": "#15202b",
    "titleBar.inactiveBackground": "#feb12d99",
    "titleBar.inactiveForeground": "#15202b99"
  }
}
```
